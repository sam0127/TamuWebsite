LED Backlight

This project was completed for my AP Computer Science class in high school. I created a dynamic backlight for my computer monitor. I created a program in Processing that took screen data and averaged the color of the border of the current display. Then I sent that data serially to an Arduino Uno micro-controller. The Arduino used that color data to change that colors on an LED strip that wrap around the back of my monitor. Here are more details of this project. Here is a video of it in action.
